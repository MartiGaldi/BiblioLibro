Per installare questa applicazione è necessario avere una versione recente di php (>= php 7), un database (consigliamo phpmyadmin) ed un utente amministratore che abbia username e password preinpostate.
Durante la prima istallazione dell'applicazione questa eseguirà i controlli sulla versione di php e mostrerà una form di richiesta per username e password di un account amministratore dei database (si può invece scegliere a piacere il nome del db)
Tra tutti gli account demo inseriti c'è come esempio di:
